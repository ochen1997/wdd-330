Every JavaScript environment has a global variable. Any variable that are created in the global scope are actually properties of this object, and any functions are mthods of it.

In a browser environment the global object is the window object, which represent the browser window that contains a web page.

The Browser Object Model is a collection of properties and methods that contain information about the browser and computer screen.

Global variables are variables that are created without using the const, let or var keywords. Global variables can be accessed in all parts of the program.

Global varaibles are actuaul properties of a global object. In a browser environment, the globle objet is the window object.

The window.alert() method will pause the execution of the program and display a message in the dialog box. The message is provided as an argument to the method, and undefined is always returned.

The window.confirm() method will stop the execution of the program and display a confirmation dialog that shows amessege as an argument and giving an option of ok or cancel. It returns the boolean values true if the user clicks ok and false if the user clicks cancel.

The window.prompt() method will stop the execution of the program. it displays a dialog that shows a message provided as an argument, as well as an input field that allows the user to enter text. This text is then return as astring when the user clicks ok. if the user clicks cancel, null is returned.

The window object has a navigator property that returns a reference to the navigator object.

The navigatorobject contains information about the browser being used.it is userAgent property will return information about the browser and operating systm being used.

The window.location property is an oblect that contains information about the URL of the current page. it contains a number of properties that provide information about different fragments of the URL. The URL property returns the full URL as a sring.

The protocol property returns a string describing a protocol eg;

  • http:
  • https:
  • pop2:
  • ftp:

The host property returns a string describing the domain of the current URL and the port number.

The hostname property returns asring describing the domain of the current URL.

The port property returns a string describing the port number, although it will return an empty string if the port is not explicity stated in the URL.

The pathname property returns a string of the path that follows the domain.

The search property returns a string that stsrts with with a "?"

The hash property returns a string that starts with a "#"

The origin property returns a string that shows the protocol and the domain where the current page orinated from.

The window.location object has the following methods;

  • reload() method- can be used to force a reload of the current page. if its given parameter of true.
  • assign() method- can be used to load another resources from a URL provided as a parameter.
  • The toString() method returns a string containing the whole URL.

The window.history property can be used to access information about any previous visited pages in the current browser session.

The window.history.length property shows how many pages have been visited before arriving at the current page.

The window.history.go() method can be used to go to a specific page, where 0 is the current page.

The window.history.forward() and window.history.back() methods that can be used to navigate forwards and backwards by one page respectively.

Anew window can be opened using the window.open() method.

The close() method can be used to close a window.

It is possible to move a window using the window.moveTo() method.

You can resize a window using the window.resizeTo() method.

The window.screen object contains information about the screen the browser is displayed on.

The availHeight and availWidth can be used to find the height and the width of the screen.

The colorDepth property can be used tom find the color bit depth of the user's monitor.

The write() method simply writes a string of text to the page. if a page is already loaded, it will completely replace the current document.

The document.write() method can also be used within a document inside script tags to inject a string into the markup. This will overwrite the rest of the HTML on the page.

To create a cookie, you assign it to javaScript's cookie jar using the document.cookie property.

Adding the string ; secure to the end of a cookie will ensure it is only transmitted over a seure HTTPS networks.

The data attribute is a way of embedding data in a web page using custom attributes that are ignored by the browser.

The names of these attributes can be decided by the developer, but they must use the following format

  • Start with data-
  • Contain only lowercase letters, numbers, hyphens, dots, colons or underscores
  • Include an optional string value

The HTML5 speifications contains a number of APIs that can help to gain access to hardware such as

  • cameras
  • batteries
  • geolocation
  • graphics card

The web Storage API provides a key-value store on the clients computer that is similar to using cookies but has fewer restrictions, more storage capacity, and generally easy to use.

If a browser supports the web Storage API the window object will have ap property called localStorage.

Everytime a value is saved to the local storage a storage event is fired.

The geolocation API is used to obtain the geographical position of the device.

If geolocation is available, it will be a property of the navaigator object. This property has a method called such as

The clearWatch() method can be used to stop the callback being called, using the ID of the watch as an argument

The Geolocation API provides a useful interface for adding location based information to a wbsite or application

The service worker API allows a worker script to run in the backgroundwith the added benefits of being able to intercept network requests.

HTML5 introduced the audio and vedio tags used to insert audio and video clips into a web page. it also introduced a media API for controlling the playback of the clips using JavaScript.

An audio clip can be inserted into a page with the audio tag, using the src attribure to point to the audio file

Adio and Video elements have anumber of properties and methods to control the playback of the clip

  • The play() method will start the clip by playing from its current position
  • The pause() method will pausse the clip at its current position
  • The volume property is anumber that can be used to set the audio volume - video.volume = 0.9;
  • The muted property is a boolean value that can be used to mute the audio
  • The currentTime property is a number value that can be used to jump to another part of the clip
  • The playbackRate property is used to fast-forward or rewind the clip by changing its value
  • The loop property is a boolean value that can be set to true to make the clip repeat in a loop.
  • The duration property can be used to see how long the clip lasts.
