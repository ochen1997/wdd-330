An object literal in JavaScript is a self contained set related values and functions. They act as a collection of named properties that map to any Javascript value such as strings, numbers, booleans, arrays and functions. if a property value is a function know as method.

An object iteral is an object that is created directly in the language by wrapping its properties in curly braces {}. Object literals are a distingushing features of the JavaScript language, as they allow objects tobe created quickly without need for defining a class.

To create an object literal, simply enter a pair of curly braces.

You can access the properties of an object using the dot notation, this will return a avalue of that property.

  • superman.name

You can also access an object's properties using bracket notation, the property is represented by a string inside square brackets, so need to be placed inside single or double quotation marks:

  • superman['name']

To call an object's method we can also use dot or bracket notation. calling a method is the same as invoking a function, do parentheses need to be placed after the method name.

The in operator can be used to check wether an object has aparticular property.

Another way is to use hasOwnProperty()method.

We can loop through all of an object's properties and methods by using a for in loop, eg;

  • (const key in superman){ console.log(key + ":" + superman[key]); }

New properties and and methods can be added to objects at anytime in a program. this is done simply by assigning avalue to the new property. for example;

  • superman.city = 'Metropolis'; output: 'Metropolis'

You can change the value of an object's properties at anytime using assignment. for example we can the value of the "real name" like this.

  • superman['real name'] = 'kal-El'; output: 'kal-El'

Any property can be removed from an object using the delete operator. for example if we wanted to remove the fly method from the superman object, we would enter the following.

  • delete superman.fly output: true

It is even possible for an object to contain other objects. Thes are know as nested objects. For example.

An object literal can be passed as a parameter to a function. This is useful when there are a large number of parameters, as it allows the arguments to be provided by name and in any order. for example;

  • function greet(greeting, name, age){ return `${greeting}! My name is ${name} and i am ${age} years old.`; }

Date object contain information about dates and time. Each object represents a single moment in time.

A constractor is used to create a new date object using the new operator.

The properties odf date object are unable to be viewed or changed directly. instead they have a number of methods known as getter methods, which returns information about the date object, such as month and year.

The getDay() and getUTCDay() methods are used to find the day of the week thta the date object falls on. it returns a number, string at 0 for sunday, up to 6 for Saturday;

The getDate() and getUTCDate() methods retuns the day of the month for the date object (note that these values start counting from 1, not 0 so they return the actual day of the month)

The getMonth() and getUTCMonth() methods canbe used to find the month of the date object it returns an integer, but remember to count from 0; so january is 0 Febuary is 1 and so on up to december being 11.

The getFullYear() and getUTCFullYear() methods return the year of the date object.

The Document Object Model represents an HTML document as a network of connected nodes that fornm a tree like structure.

The DOM treats everything on a web page as a node. HTML tags, the text inside these tags, even the atrributes of a tag are all nodes.

The DOM provides several methods that allow us to access any element on a page. These methods will return a node object or node list, which is an array like object. These objects can then be assigned to a variable and be inspected or modified.

The getElementById() method does exactly what it says on the tin. it returns a reference to the element with a unique id atribute thta is given as an argument.

The getElementByTagName() will return a live node list of all the elements with the tag name that is provided as an argument.

The getElementByClassName() will return a live node list of all elements that have the class name that is supplied as an argument.

The document. querySelector() method allows you to use CSS notation to find the first element in the document that matches a CSS selector provided as an argument.

The document.querySelectorAll() method also uses CSS notation but returns a node of list odf all the elements in the document that match the CSS query selector.

The getAttribute() method returns the value of the attribute provided as an assignment.

The remove() method will remove as specific class from an element. For example, we could remove the class of 'warrior' with the following code.

  • wonderWoman.ClassList.remove('warrior')

The toggle method is aparticularly useful method that will add a class idf an element does not have it already, and remove the class if it does have it for example,

  • wonderWoman.ClassList.toggle('hero') output: false
  • wonderWoman.ClassList.toggle('sport') output: true
